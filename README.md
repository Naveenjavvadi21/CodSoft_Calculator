# Basic Calculator

## Description

Basic Calculator is a web-based application developed using HTML, CSS, and JavaScript. It provides an interactive interface for performing arithmetic operations such as addition, subtraction, multiplication, and division. The calculator features a clean design, responsive layout, and real-time calculation functionality, making it an excellent project for learning front-end web development.

## Features

- Addition Operation
- Subtraction Operation
- Multiplication Operation
- Division Operation
- Clear Display Function
- Responsive User Interface
- Interactive Button Controls
- Real-Time Calculation
- CSS Grid Layout for Button Alignment

## Technologies Used

- HTML5
- CSS3
- JavaScript

- Learn HTML page structure.

## Calculator Functions

| Button | Function |
|----------|----------|
| 0-9 | Number Input |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| = | Calculate Result |
| C | Clear Display |

## Working Principle

1. User clicks number buttons.
2. Selected values appear on the display screen.
3. Mathematical operators are selected.
4. JavaScript processes the expression.
5. Clicking "=" calculates and displays the result.
6. Clicking "C" clears the display.
